# POSfromMolt OAuth Consent Pages

This repository contains the required pages for Google OAuth consent screen for personal use of POSfromMolt with Gmail API.

## Purpose

These pages fulfill Google's OAuth verification requirements for accessing Gmail API when using POSfromMolt for email management tasks.

## Pages Included

- Privacy Policy
- Terms of Service
- Application Information
